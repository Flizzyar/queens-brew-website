# Queens Brew ☕♟️

A static café website concept for **Queens Brew** — a chess-inspired coffee shop.  
Built with **HTML and CSS** with multiple pages (Home + Menu).

---

## ✨ Features

- Home page with welcome section, opening hours and contact info
- Separate menu page
- Responsive layout (mobile-friendly)
- Custom images and typography (assets included)

---

## 🧱 Tech Stack

- HTML
- CSS

---

## ▶️ Run Locally

No installation needed.

1. Clone the repository:
   ```bash
   git clone https://github.com/Flizzyar/queens-brew-website.git
   ```
2. Open `index.html` in your browser.
